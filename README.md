Landing Page – Voyages à Vélo

Projet réalisé dans le cadre du parcours Foundations de The Odin Project.
L’objectif était de reproduire une landing page en utilisant HTML et CSS, en mettant l’accent sur les bases de la mise en page, Flexbox, et la structuration d’un contenu marketing simple.

Objectifs du projet:
Structurer une page web en HTML
Utiliser Flexbox pour créer une mise en page moderne
Apprendre à gérer des images, du texte et des boutons
Travailler la cohérence visuelle (couleurs, typographie, espacements)

Technologies utilisées
HTML5
CSS3
Flexbox
Google Fonts

Fonctionnalités
Header avec liens externes
Section héro avec image
Section “Conseils” sous forme de cards alignées
Citation stylisée
CTA clair (Call To Action)
Footer simple
